This example demonstrates different transforms.Change the transform of text by clicking on the different transform of text to see what happend!
the code:
div {
text-transform:none;
}

Let's play with text transform!

none No capitalization. The text renders as it is. This is default.
capitalize Transforms the first character of each word to uppercase.
uppercase Transforms the first character of each word to uppercase.
lowercase Transforms all characters to lowercase.
initial Sets this property to its default value.
