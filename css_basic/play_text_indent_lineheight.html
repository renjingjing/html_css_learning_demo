This example demonstrates different indents and line-height.Change the indent or line-height of text by clicking on the different indent or line-height of text to see what happend!

the code:
div {
text-indent:0px;
line-height:normal;
}

Let's play with text indent and line-height!

text indent
10px 25px 100px 1cm 2cm Length Defines a fixed indentation in px, pt, cm, em, etc. Default value is 0.
10% -10% % Defines the indentation in % of the width of the parent element.
initial initial Sets this property to its default value.
text line-height
normal normal A normal line height. This is default.
2 3.5 number number A number that will be multiplied with the current font size to set the line height.
0px 10px 30px length A number that will be multiplied with the current font size to set the line height.
50% 120% % A line height in percent of the current font size.
initial initial Sets this property to its default value.
