This example demonstrates different vertical-aligns.

the dode:
span{
vertical-align:initial;
}

Let's play with vertical align!

baseline Align the baseline of the element with the baseline of the parent element. This is default.
sub Aligns the element as if it was subscript.
super Aligns the element as if it was superscript.
top The top of the element is aligned with the top of the tallest element on the line.
text-top The top of the element is aligned with the top of the parent element's font.
middle The element is placed in the middle of the parent element.
bottom The bottom of the element is aligned with the lowest element on the line
text-bottom The bottom of the element is aligned with the bottom of the parent element's font.
0px 10px 25px -25px -50px Length Raises or lower an element by the specified length.
1cm Length
10% Raises or lower an element in a percent of the "line-height" property.
50% Raises or lower an element in a percent of the "line-height" property.
initial Sets this property to its default value.
